| 功能              | 示例代码                                                             | 效果               |
| --------------- | ---------------------------------------------------------------- | ---------------- |
| 输出 `true/false` | `std::cout << std::boolalpha << flag << '\n';`                   | `true` / `false` |
| 输出 `1/0`        | `std::cout << std::noboolalpha << flag << '\n';`                 | `1` / `0`        |
| 固定小数格式          | `std::cout << std::fixed << pi << '\n';`                         | `3.141593`       |
| 指定小数位数          | `std::cout << std::fixed << std::setprecision(2) << pi << '\n';` | `3.14`           |
| 科学计数法           | `std::cout << std::scientific << pi << '\n';`                    | `3.141593e+00`   |
| 十六进制输出          | `std::cout << std::hex << 255 << '\n';`                          | `ff`             |
| 八进制输出           | `std::cout << std::oct << 255 << '\n';`                          | `377`            |
| 十进制输出           | `std::cout << std::dec << value << '\n';`                        | 恢复十进制            |
| 设置输出宽度          | `std::cout << std::setw(5) << 42 << '\n';`                       | `42`             |
| 设置填充字符          | `std::cout << std::setfill('0') << std::setw(5) << 42 << '\n';`  | `00042`          |
| 左对齐             | `std::cout << std::left << std::setw(10) << "abc" << '\n';`      | `abc`            |
| 右对齐             | `std::cout << std::right << std::setw(10) << "abc" << '\n';`     | `abc`            |
| 显示正号            | `std::cout << std::showpos << 42 << '\n';`                       | `+42`            |
| 取消正号            | `std::cout << std::noshowpos << 42 << '\n';`                     | `42`             |
| 大写十六进制          | `std::cout << std::uppercase << std::hex << 255 << '\n';`        | `FF`             |
| 小写十六进制          | `std::cout << std::nouppercase << std::hex << 255 << '\n';`      | `ff`             |
